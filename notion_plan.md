# 평가 관리

---

## 개요

본 프로젝트는 조직 정보와 평가 기준을 관리하고, 사원별 성과·역량 평가, 다면 평가, 면담, 최종 등급 확정을 수행할 수 있는 사원 평가 프로그램입니다. 또한 평가 결과를 시각화하여 부서별 등급 분포와 사원별 평가 점수를 효율적으로 확인할 수 있도록 구성하였습니다.

## 요구 사항

### 업무 흐름도 요약

- 부서원: 본인 평가 ⇒ 부서장 평가 ⇒ 평가 확정 (최종 평가자 확정)
- 부서장: 본인 평가 ⇒ 다면 평가 ⇒ 평가 확정 (최종 평가자 확정)

### 메뉴별 요구 사항

- 홈 화면 (로그인 화면)
    
    **화면 목적**
    
    - 등록된 사원 정보를 바탕으로 로그인 할 수 있어야 합니다.
    - 해당 사원 정보(에 따라 접근 가능한 메뉴가 관리되어야 합니다.
    
    **주요 기능**
    
    - 사번/비밀번호 인증, 로그인 상태 유지, 비밀번호 초기화 등.
    - 비밀번호 5회 오류 시 계정 잠금 기능.
    - 퇴사 상태인 경우 로그인 제한 메시지 출력.

- 관리 페이지 > 부서 관리
    
    **화면 목적**
    
    - 조직의 부서 정보를 생성/관리 합니다.
    - 관리자가 부서장을 지정/변경 할 수 있습니다.
    
    **주요기능**
    
    - 부서 계층 구조 설정(상위 부서), 부서장 매핑, 사용 여부 관리.
    
    | **구분** | **상세 설명** | **비고** |
    | --- | --- | --- |
    | **조회 조건** | • 부서명(텍스트 또는 부서 코드 검색), 사용 여부(전체/사용/미사용).
    • 기본값: 전체 조회. |  |
    | **부서 기본 정보** | • **부서 코드**: 시스템 자동 생성 (PK).

    • **상위 부서**: 조직 계층 구조 설정을 위한 셀렉트 박스. |  |
    | **리더(부서장) 지정** | • 해당 부서에 소속된 사원 중 1명을 선택하여 리더로 등록.
    • 리더는 언제든지 변경(수정) 가능하며, 평가자 생성 프로세스 시 평가자로 자동 매핑됨. |  |

    | **입력 유효성 (Validation)** | • 필수 항목(부서명, 사용여부) 누락 시 저장 방지 알림.
    • 사원은 반드시 하나의 부서에만 소속되어야 함(중복 소속 불가 로직). |  |
- 사원 관리
    
    **화면 목적**
    
    - 사원의 입사부터 퇴사까지의 신상 정보와 재직 상태를 기록하고 관리함.
    
    **주요 기능**
    
    - 사원 정보 등록/수정, 부서 배정, 직급 관리, 재직 상태(재직/휴직/퇴사) 변경.
    
    **요구 기능 상세**
    
    | **구분** | **상세 설명** | **비고** |
    | --- | --- | --- |
    | **조회 조건** | • 사번, 성명(텍스트), 부서(트리/셀렉트), 재직 상태(전체/재직/휴직/퇴사).
    • 기본값: 재직 중인 전체 사원 조회. |  |
    | **인적 사항 관리** | • **사번**: 시스템 고유 번호 (ID로 활용, 수정 불가).
    • **성명**: 사원 이름.
    • **이메일/핸드폰**: 연락처 정보. |  |
    | **소속 및 직급** | • **부서**: 부서 관리 메뉴에 등록된 부서 중 하나를 선택 (중복 소속 불가). |  |
    | **상태 및 일자** | • **입사일**: 필수 입력 (근속 연수 및 평가 대상 기준).
    • **재직 상태**: 재직, 휴직, 퇴사 등 (상태에 따라 로그인 권한 및 평가 대상 여부 결정). |  |
    | **데이터 제약 사항** | • **부서장 확인**: 부서 관리에서 리더로 지정된 사원은 부서 변경 시 리더 권한 이양 확인 절차 필요.
    • **퇴사 처리**: 퇴사일 입력 시 시스템 로그인 권한 즉시 차단 로직 적용. |  |
    | **입력 유효성 (Validation)** | • 사번 중복 체크 필수.
    • 필수 항목(이름, 입사일, 부서) 미입력 시 저장 불가 처리. |  |

    - 평가자 자동 생성 기능을 지원합니다.
    
    **주요 기능**
    
    - 조직별 평가자 일괄 생성, 평가자 수동 수정
    
    **요구 기능 상세**
    
    | **구분** | **상세 설명** | **비고** |
    | --- | --- | --- |

    | **자동 생성 로직** | • **1차 평가자**: 해당 사원의 소속 부서장(Leader)으로 매핑.
    • **최종 평가자**: 시스템 설정에 따른 특정 직급자 또는 임원 매핑. |  |
    | **수동 수정 기능** | • 자동 생성된 평가자를 관리자가 변경 가능.
    • **예외 처리**: 퇴사자의 경우 평가 제외. |  |
    | **무결성 체크** | • **자기 평가 방지**: 부서장의 경우 본인을 평가하지 않도록 제어. |  |
    | **일괄 처리** | • 선택한 부서 전체에 대해 평가자 생성/초기화 기능.
    • (선택기능) 엑셀 업로드를 통한 평가자 관리 |  |
- 평가요소 관리
    
    **화면 목적**
    
    - 성과/역량/다면/면담 등 각 평가 단계에서 사용할 세부 항목을 정의
    - 최종 등급 산출을 위한 평가 유형별 비중(가중치)을 관리함.
    
    **주요 기능**
    
    - 평가 유형별 가중치 설정(성과 40, 역량 60 등), 평가 문항(지표) 등록 및 수정, 다면/면담 가이드라인 설정.
    
    **요구 기능 상세**
    
    - **유형 관리**: 관리하고자 하는 평가 유형(성과, 역량, 다면, 면담 등)별 항목 관리.
    - **비중 설정**: 성과와 역량의 합이 100%가 되도록 가중치를 입력함.

- 성과/업무 평가 관리
    
    **화면 목적**
    
    - 임직원이 본인의 성과 및 업무 평가 상세 내역을 조회
    - 평가 항목별 수행 과정, 성과, 잘된 점 및 개선점 등에 대한 구체적인 피드백 공유.
    
    **주요 기능**
    
    - **평가 이력 조회:** 기준년도 및 평가기간 설정을 통한 대상 문서 검색.
    - **평가 대상자 정보 표시:** 부서, 직위, 직책, 직종 등 대상자의 기본 정보 확인
    - **상세 평가 내역 확인:** 주요 수행 업무별 목표, 배점, 수행 과정 및 결과에 대한 상세 텍스트 피드백 조회.
    - **평가 등급 및 점수 산출:** 종합 평가 및 본인 평가 점수 확인.
    
    **요구 기능 상세**
    
    - 1차 평가는 개인이 직접 본인 평가 진행.
    - 부서장의 경우 부서원 평가를 진행. (부서원 평가 이전 진행 불가)
    - 성과 평가 피드백과 종합 평가 영역 존재.

- 다면 평가

- 면담 관리
    
    **화면 목적**
    
    - 정기/수시 면담 내용을 기록하여 사원의 성장을 지원하고 향후 평가 시 참조로 사용하는 화면
    
    **주요 기능**
    
    - 평가자(부서장)가 본인의 부서원 리스트 중 면담을 진행할 대상자를 선택함.
    - 면담 일자, 장소 및 '평가요소 관리'에서 정의된 면담 주제별로 대화 내용을 입력함.
    - 작성된 내용을 저장하면 피평가자가 내용을 확인할 수 있음.
    
    **요구 기능 상세**
    
    - 부서장 권한: 소속 부서원 리스트 출력.
    - 사원 권한: 본인의 면담 기록 목록만 조회 가능.
    - 면담 구분: 정기 면담, 수시 면담 등
    - 면담 일시: 면담이 진행된 날짜와 시간 기록.
    - 면담 주제: '평가요소 관리'에서 설정한 주제(업무 애로사항, 경력 개발 등)가 라벨로 자동 노출.
    - 상세 내용: 주제별 면담 내용을 입력창으로 기록
    
- 최종 등급 확정
    
    **화면 목적**

    **주요 기능**
    
    - 조회: 담당 부서의 평가 항목에 대해 조회.

    - 등급 조정: 산출된 점수 기반으로 자동 부여된 등급을 확인하고, 필요 시 예외적으로 등급을 조정함.

    - 조직의 임원진에 한해 접근 가능.
    - 최종 확정 이전에 등급을 최종 조정할 수 있음.
    - 확정 시 최종 등급을 계산하여 저장.

- 사원별 평가 점수와 부서별 등급 현황
    
    **화면 목적**
    
    - 피평가자들의 성과업무평가, 역량평가 점수, 종합 점수, 평가 등급을 확인하기 위한 화면
    
    **주요 기능**
    
    - 사번, 부서를 기준으로 조회
    - 로그인중인 사번의 권한을 기준으로 사이드바에서 노출 여부를 결정
    
    **요구 기능 상세**
    
    - 부서장인 경우에만 본인의 부서원에 대한 최종 평가 결과를 조회
    - 평가자인 경우에는 본인의 피평가자 전체를 조회 가능 (부서별 조회 기능 포함)
    - 성과업무평가, 역량평가 점수 및 평가기준을 바탕으로 종합 점수 및 평가 등급을 계산
    - 로그인중인 사번의 권한이 부서장이 또는 평가자가 아닌 경우에는 사이드바에 노출시키지 않음
    - 피평가자 리스트는 직책 기준으로 정렬
